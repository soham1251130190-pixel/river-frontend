The Gap We're Filling

Problem & Solution

⚡ The Problem
🗺️
CPCB has ~1,000 monitoring stations for 14,000 km of Indian rivers — zero coverage of minor water bodies, canals, and local streams.
💸
Government monitoring stations cost ₹1–5 lakh per installation. Impractical to scale to the thousands of unmonitored locations.
⏰
Manual sampling is done weekly or monthly. Industrial dumping events happening at 3 AM go completely undetected.
👁️
TDS contamination from chemicals and heavy metals is invisible to the naked eye — no colour change, no smell, no visible sign.
✅ Our Solution
💧
Deploy compact sensor nodes at under ₹1,300 per location — 100x cheaper than government stations. Scalable to hundreds of sites.
🤖
Isolation Forest ML model trained on CPCB clean water baseline data. Detects abnormal multi-parameter combinations that threshold rules miss.
📱
WhatsApp alerts delivered within 60 seconds of anomaly detection — reaching municipal authorities directly on phones they already use.
🌐
Public dashboard with real-time map and 24-hour history accessible to citizens, journalists, and local governments — fully open.
Complete Technology Stack

How It's Built

⚙️
ARDUINO UNO
Microcontroller that reads all three sensors via CD4051 multiplexer and sends JSON to ESP-01 every 30 seconds.
Hardware
📶
ESP-01 + HIVEMQ
ESP-01 Wi-Fi module publishes sensor data to HiveMQ cloud MQTT broker over TLS. Handles reconnection automatically.
Connectivity
🐍
FLASK (PYTHON)
Python backend subscribes to MQTT, runs ML scoring on every reading, stores to InfluxDB, and serves REST API endpoints.
Backend
🗄️
INFLUXDB
Time-series database storing every reading with nanosecond precision. Enables fast range queries for history graphs.
Database
🤖
ISOLATION FOREST
scikit-learn anomaly detection model trained on CPCB baseline data. Scores each reading 0.0–1.0. Threshold: 0.60.
ML Model
📱
TWILIO WHATSAPP
Automated WhatsApp alerts with TDS value, temperature, pH, anomaly score, and severity delivered within 60 seconds.
Alerts
🗺️
LEAFLET.JS
Interactive OpenStreetMap-based map showing sensor node with real-time colour-coded status marker. Free, open-source.
Frontend
📊
CHART.JS
Responsive line charts for 24-hour TDS, temperature, pH, and anomaly score history. Animated, interactive tooltips.
Frontend
🚀
RAILWAY + VERCEL
Flask backend deployed on Railway free tier. This dashboard deployed on Vercel free tier. Both provide public HTTPS URLs.
